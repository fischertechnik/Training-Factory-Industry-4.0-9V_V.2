# fischertechnik Training Factory Industry 4.0 9V V.2
This project contains the ROBO Pro Coding programs for the fischertechnik [**Training Factory Industry 4.0 9V V.2**]([https://www.fischertechnik.de/en/service/elearning/teaching/lernfabrik-4). The library requires at least the [TXT firmware version 4.5.1](https://github.com/fischertechnik/FT-TXT/releases](https://www.fischertechnik.de/en/industry-and-universities/technical-documents/simulieren/lernfabrik-4-0-9v-v-2)).
